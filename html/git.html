Git

  • git -v - выводит версию Git, если он правильно установлен.
  • git init - инициализация репозитория.
  • git clone /ссылка/ - клонирует репозиторий по указанной ссылке.
  • git status - отображает изменения в репозитории.
  • git add /название файла/ - добавляет файл в отслеживаемые.
  • git commit -m /название коммита/ - добавление коммита.
  • git push - внести изменения в репозиторий.
  • git log - выводит список всех коммитов.
  • git pull - подтягивает изменения из удалённого репозитория в локальный.
